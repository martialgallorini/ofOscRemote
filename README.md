ofOscRemote
===========

Simple interface to sync 3 VDMX players via OSC

Developped with openFrameworks 0.8.4 under MacOSX Mavericks and XCode 6

Core oF addons :
- ofxNetwork
- ofxOsc

External addons :
- ofxUI from Reza Ali - https://github.com/rezaali/ofxUI
